# sin

Calculate the sine of an argument

## Syntax

```
y = sin[radians]
y = sin[degrees]
```

## Attributes

- radians
- degrees

## Description

Calculates the sine of an input in radians or degrees. `Sin` operates element-wise on its inputs.

## Examples

```
y = sin[degrees]

```
