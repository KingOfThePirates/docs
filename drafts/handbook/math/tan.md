# tan

Calculate the tangent of an angle

## Syntax

```
y = tan[radians]
y = tan[degrees]
```

## Attributes

- radians - the angle in radians
- degrees - the angle in degrees

## Description

Calculates the tangent of an input in radians or degrees. `Sin` operates element-wise on its inputs.
