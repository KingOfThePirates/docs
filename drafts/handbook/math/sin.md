# sin

Calculate the sine of an angle

## Syntax

```
y = sin[radians]
y = sin[degrees]
```

## Attributes

- radians - the angle in radians
- degrees - the angle in degrees

## Description

Calculates the sine of an input in radians or degrees. `Sin` operates element-wise on its inputs.
