# cos

Calculate the cosine of an angle

## Syntax

```
y = cos[radians]
y = cos[degrees]
```

## Attributes

- radians - the angle in radians
- degrees - the angle in degrees

## Description

Calculates the cosine of an input in radians or degrees. `Cos` operates element-wise on its inputs.
