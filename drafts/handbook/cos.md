# cos

Calculate the cosine of an argument

## Syntax

```
y = cos[radians]
y = cos[degrees]
```

## Attributes

- radians
- degrees

## Description

Calculates the cosine of an input in radians or degrees. `Cos` operates element-wise on its inputs.

## Examples

```
y = cos[degrees]

```
